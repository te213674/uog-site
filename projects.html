Уфа — Установка УОГ с газопоршневой генерацией

ПНГ Уфа

Комплексная установка очистки попутного нефтяного газа с последующей генерацией электроэнергии. Проект включал монтаж УОГ, газопоршневой установки и систем автоматики.

УОГ-5000 — Крупная промышленная установка

5000 нм³/час Промышленная

Крупнейшая установка в модельном ряду УОГ. Предназначена для очистки больших объёмов газа на промышленном объекте. Включает полный цикл очистки от сероводорода.

УОГ-1500ВД — Высоконапорная версия

8 МПа ВД Высокое давление

Специализированная версия установки для работы при высоком давлении (до 8 МПа). Предназначена для месторождений с высоким пластовым давлением.

УОГ-100 — Стандартная установка

100 нм³/час Стандарт

Установка средней производительности для очистки попутного газа. Оптимальное решение для небольших месторождений.
