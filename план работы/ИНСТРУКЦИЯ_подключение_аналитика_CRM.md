# 📋 Инструкция: Подключение аналитики и CRM к сайту SorbGaz

## 1. Яндекс.Метрика (создание счётчика)

1. Перейдите на https://metrica.yandex.ru/
2. Нажмите **«Добавить счётчик»**
3. **Название**: `SorbGaz — УОГ`
4. **Адрес сайта**: `https://www.sorbgaz.ru/`
5. В настройках включите:
   - ☑️ Карта кликов (Webvisor)
   - ☑️ Анализ форм
   - ☑️ Сбор данных электронной коммерции (для будущего)
   - ☑️ Отложенный код (асинхронный)
6. После создания скопируйте **номер счётчика** (8 цифр, например `98765432`)
7. Замените в коде сайта `XXXXXXXX` на ваш номер счётчика во всех файлах

### Цели для настройки (в Метрике → Настройки → Цели):
| Имя цели | Условие | Тип |
|----------|---------|-----|
| FORM_SUBMIT_MAIN | Событие: reachGoal → FORM_SUBMIT_MAIN | JavaScript |
| FORM_SUBMIT_ANALYSIS | Событие: reachGoal → FORM_SUBMIT_ANALYSIS | JavaScript |
| CLICK_PHONE | Клик по ссылке с tel: | JavaScript |
| CLICK_EMAIL | Клик по ссылке с mailto: | JavaScript |

## 2. Google Analytics 4 (создание ресурса)

1. Перейдите на https://analytics.google.com/
2. **Администратор** → **Создать ресурс** → **Поток данных** → **Веб**
3. **URL сайта**: `https://www.sorbgaz.ru/`
4. **Название потока**: `SorbGaz — Основной`
5. Скопируйте **Measurement ID** (формат `G-XXXXXXXXXX`)
6. Замените в коде сайта `G-XXXXXXXXXX` на ваш ID

### События GA4 (настраиваются автоматически через gtag):
- `FORM_SUBMIT_MAIN` — отправка основной формы
- `FORM_SUBMIT_ANALYSIS` — отправка формы анализа газа

## 3. Bitrix24 (интеграция форм)

### Шаг 1: Создание вебхука в Битрикс24

1. Откройте ваш портал Битрикс24 (например, `https://ВАШ_ДОМЕН.bitrix24.ru/`)
2. Перейдите: **Приложения** → **Другие приложения** → **Вебхуки**
3. Нажмите **«Добавить вебхук»** → **«Входящий вебхук»**
4. Придумайте название: `SorbGaz Сайт`
5. В правах доступа выберите:
   - ☑️ **CRM** (лиды): `crm.lead.add`, `crm.lead.fields`
6. Сохраните — вы получите URL вида:
   ```
   https://ВАШ_ДОМЕН.bitrix24.ru/rest/12345/abcdefg...
   ```
7. Замените в коде сайта `BITRIX24_CONFIG.webhookUrl` на этот URL

### Шаг 2: Что делает интеграция

При отправке формы на сайте автоматически создаётся **Лид** в Битрикс24 со следующими полями:
- **Заголовок**: «Заявка с сайта SorbGaz: [Имя]»
- **Имя**, **Телефон**, **Email**
- **Компания**
- **Комментарий** (сообщение из формы)
- **Источник**: WEB
- **UTM-метки** (если есть в URL)

### Шаг 3: Проверка

1. Заполните форму на сайте и нажмите «Отправить»
2. Зайдите в Битрикс24 → **CRM** → **Лиды** — должен появиться новый лид
3. Если не появился — откройте консоль браузера (F12) и проверьте ошибки

## 4. Google Search Console (индексация)

1. Перейдите на https://search.google.com/search-console/
2. Добавьте ресурс: `https://www.sorbgaz.ru/`
3. **Подтверждение права собственности**:
   - Выберите способ «Мета-тег»
   - Добавьте тег `<meta name="google-site-verification" content="...">` в `<head>` index.html
4. После подтверждения загрузите sitemap.xml:
   - **Файлы Sitemap** → **Добавить новую карту сайта** → `https://www.sorbgaz.ru/sitemap.xml`

---

## 5. Яндекс.Вебмастер (индексация для Яндекса)

1. Перейдите на https://webmaster.yandex.ru/
2. Добавьте сайт `https://www.sorbgaz.ru/`
3. Подтвердите права (мета-тег или TXT-запись DNS)
4. **Индексирование** → **Файлы Sitemap** → добавьте `https://www.sorbgaz.ru/sitemap.xml`
5. **Переходы** → проверьте robots.txt
