## 3. ИНТЕРАКТИВНЫЕ ЭЛЕМЕНТЫ

### 3.1. Калькулятор экономии

**Функционал:**
- Ввод производительности (нм³/час)
- Ввод содержания H2S (%)
- Ввод часов работы в год
- Выбор технологии для сравнения
- Расчёт годовой экономии

**Формула:**
```
Годовой объём газа = Производительность × Часы работы
Затраты УОГ = Годовой объём × 1,35 руб/нм³
Затраты конкурента = Годовой объём × (себестоимость конкурента)
Экономия = Затраты конкурента - Затраты УОГ
```

**Техническая реализация:**
- JavaScript (client-side)
- Сохранение результатов в URL (для шеринга)
- CTA: «Получить подробный расчёт» (отправка на email)
