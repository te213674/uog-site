## 3. О ТЕХНОЛОГИИ

### 3.1. Принцип работы (How It Works)

**Структура страницы:**

1. **Введение** — что такое жидкофазное окисление
2. **Схема процесса** — инфографика с 5 стадиями
3. **Химические реакции** — для технических специалистов
4. **Преимущества технологии** — сравнение с аналогами

**5 стадий очистки (для инфографики):**

```
┌─────────────────────────────────────────────────────────┐
│  1. Механическая очистка → 2. Абсорбция H2S            │
│     (скруббер)              (колонны 2-4 шт.)          │
│                          ↓                               │
│  5. Фильтрация ← 4. Регенерация ← 3. Сепарация         │
│     (отстойник,      (окисление воздухом)  (каплеуловитель)
│      пресс-фильтр)                                      │
└─────────────────────────────────────────────────────────┘
```

**Химические реакции:**
```
Поглощение:    H2S(г) + Cat(окс) → Cat(восст) + S + H2O
Регенерация:   Cat(восст) + ½O2 → Cat(окс) + H2O
Общая:         H2S + ½O2 → S + H2O
```

### 3.2. Преимущества

| Преимущество | Описание |
|--------------|----------|
| Нечувствительность к CO2 | В отличие от щелочной очистки |
| Автоматическая подача | В отличие от адсорбции на угле |
| Работа на низких давлениях | От 0,1 Бар |
| Низкая себестоимость | 1,2-1,5 руб/нм³ |
| Регенерация | Без замены реагента |
| Экологичность | Отход — элементарная сера |

### 3.3. Область применения

| Отрасль | Тип газа | Применение |
|---------|----------|------------|
| Нефтегаз | ПНГ | Подготовка для ГПУ/ГТУ |

## 5. СРАВНЕНИЕ ТЕХНОЛОГИЙ

### 5.1. Сравнительная таблица

| Технология | Себестоимость | Капзатраты | Применимость | Недостатки |
|------------|---------------|------------|--------------|------------|
| **Жидкофазное окисление (УОГ)** | 1,2-1,5 руб/нм³ | Средние | 50-10 000 нм³/час | — |
| Щелочная (Sulfurex) | 1,6-3,2 руб/нм³ | Низкие | 100-10 000 нм³/час | Зависит от CO2 |
| Аминная очистка | 2-4 руб/нм³ | Высокие | 5 000-100 000 нм³/час | Требует утилизации H2S |
| Адсорбция (уголь) | 1,5-3 руб/нм³ | Низкие | до 1 000 нм³/час | Периодическая работа |
| Оксиды железа | от 2 руб/нм³ | Низкие | до 50 нм³/час | Частая замена адсорбента |
| Мембранная | 3-5 руб/нм³ | Высокие | 1 000-50 000 нм³/час | Не для H2S |

### 5.2. Калькулятор экономии

**Входные данные:**
- Производительность (нм³/час)
- Содержание H2S (%)
- Часы работы в год

**Расчёт для УОГ:**
```
Затраты = Производительность × Часы × Себестоимость
```

**Сравнение с конкурентом:**
```
Экономия = (Себестоимость_конкурента - Себестоимость_УОГ) × Объём
```

**Пример:**
- Производительность: 200 нм³/час
- Часы работы: 8 760 (круглогодично)
- Объём газа: 1 752 000 нм³/год

| Технология | Себестоимость | Годовые затраты |
|------------|---------------|-----------------|
| УОГ | 1,35 руб/нм³ | 2 365 200 руб |
| Sulfurex | 2,5 руб/нм³ | 4 380 000 руб |
| **Экономия** | | **2 014 800 руб/год** |
