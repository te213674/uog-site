### 5.3. Калькулятор/Квиз

**Срок:** 1 неделя

**Калькулятор экономии:**
```javascript
// calculator.js
export const calculateSavings = (capacity, h2sContent, hoursPerYear) => {
  const annualVolume = capacity * hoursPerYear;
  const costUOG = annualVolume * 1.35;
  const costCompetitor = annualVolume * 2.5; // Sulfurex
  const savings = costCompetitor - costUOG;
  return { costUOG, costCompetitor, savings };
};
```

**Квиз:**
```javascript

];
```
