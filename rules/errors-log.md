## Категории ошибок

| Категория | Описание |
|-----------|----------|
| HTML | Проблемы с разметкой, семантикой, доступностью |
| CSS | Проблемы со стилями, адаптивностью, кроссбраузерностью |
| JavaScript | Ошибки в логике, обработчиках событий, асинхронности |
| Дизайн | Проблемы с визуалом, типографикой, цветами |
| Производительность | Медленная загрузка, большие файлы |
| SEO | Проблемы с мета-тегами, структурой, индексацией |

---

## Приоритеты

| Приоритет | Описание | Срок исправления |
|-----------|----------|------------------|
| Высокий | Критические ошибки, блокирующие работу | Немедленно |
| Средний | Заметные проблемы, ухудшающие UX | В течение спринта |
| Низкий | Косметические проблемы, улучшения | По возможности |

---

## Часто встречающиеся ошибки (для быстрого доступа)

### HTML
- Отсутствие alt у изображений
- Отсутствие label у форм
- Неправильная иерархия заголовков
- Отсутствие lang атрибута у html

### CSS
- Специфичность селекторов
- Отсутствие сброса отступов
- Проблемы с z-index
- Отсутствие mobile-first подхода

### JavaScript
- Необработанные исключения
- Утечки памяти (неотписанные event listeners)
- Блокировка основного потока
- Отсутствие валидации данных
