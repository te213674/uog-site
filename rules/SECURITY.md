# 🔒 Безопасность проекта

## ⚠️ ВАЖНО: GitHub Token

**Дата проверки:** 02.04.2026

### Что произошло
В файле `create-repo.ps1` был обнаружен GitHub Personal Access Token (PAT), который был запушен в публичный репозиторий.

**Токен:** `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`

### Что нужно сделать (обязательно)

1. **Перейдите на GitHub:** https://github.com/settings/tokens
2. **Найдите токен** с префиксом `ghp_`
3. **Удалите токен** немедленно
4. **Создайте новый токен** (если нужен) с минимальными правами:
   - Только `repo` для конкретного репозитория
   - Без доступа к организациям
   - Сроком действия не более 30 дней

### Профилактика
- Никогда не добавляйте токены в код
- Используйте переменные окружения (`.env`)
- Для GitHub Actions используйте Secrets
- Проверяйте `.gitignore` перед коммитами
